# Loja do Seu Manoel 🧃📦

API para otimizar o empacotamento de pedidos com base em tamanhos de caixas disponíveis.

## 🛠️ Tecnologias
- ASP.NET Core 6
- SQL Server 2019
- Docker + Docker Compose
- Swagger para testes
